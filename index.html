Just a quick overview about Mobile. So, why Mobile is so important?

By 2014, mobile internet should take over desktop internet usage.

Mobile is a constantly growing area, is reaching the desktop internet usage, and will probably surpass it by 2014. That's why is important. And that's why Add-on SDK has an experimental support for Mobile. Here I'm showing a couple of example related to mobile and SDK.

Firefox Mobile

Firefox Mobile is an Android browser. It shares the same engine of Firefox Desktop, that is Gecko, but has several differences too. Add-on SDK takes care of these differences for you: if you write an extensions using our APIs, it will works as is on both Firefox Desktop and Mobile.

You can also write add-ons specifically to address mobile's issue.

These Are Not The Droids You're Looking For.

Phony is an add-on that disguise your Firefox Mobile as another browser. It's useful especially when a certain site provide a webkit-only version of its mobile site.

Firefox OS

Firefox Mobile is not the only mobile product in Mozilla. Firefox OS, codename B2G (Boot 2 Gecko) is the new Mozilla Operating System for Mobile platform. As the code name says, it's based on Gecko too. Add-on SDK was used to create ...

R2D2B2G

R2D2B2G. What is it? You basically take R2D2, you add a cup of of B2G, and you get the "Firefox OS Simulator", codename R2D2B2G. The Firefox OS Simulator was made using Add-on SDK.

Also, because Firefox OS is based on Gecko, ideally you could use add-ons to extends the applications you have installed on your mobile device. In this sense, it will became a sort of universal plug-in system. I'm really looking forward to have it!
